# 程序设计基础 (Fundamentals of Programming)

本仓库用于记录清华大学计算机系大一《程序设计基础》课程的日常代码、课后作业以及课程大作业。

## 📖 仓库简介

通过本课程的学习与实践，逐步掌握 C++ 的核心语法与编程思想，实现从基础逻辑到复杂系统架构的初步认知。仓库内容主要涵盖以下几个核心模块：
- **基础语法与控制流**：变量、循环、条件判断等。
- **内存管理**：指针、引用（如 `int&` 的底层逻辑）、动态内存分配。
- **数据结构与算法初步**：链表、递归、BFS、DFS等的基础实现。
